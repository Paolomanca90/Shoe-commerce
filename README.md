## Getting Started



### Prerequisites 1️⃣ <span id="prerequisites"></span>

This is the base you need to use the software.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation 2️⃣ <span id="installation"></span>

Testing the project is really simple

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Generate your DB
   ```sh
   update-database
   ```

<!-- USAGE EXAMPLES -->
## Usage 3️⃣ <span id="usage"></span>

The site is divided into 2 sections: one dedicated to admin and one to users.
First of all you will have to create an admin in your DB. This way you will be able to log in and add products.
For each shoe, a presentation is displayed on the home page, while clicking on the product opens a detail page with a complete description and 3 other photos that can be viewed through a modal with a carousel inside. Obviously it is possible to add the goods to the cart and finalize the order.
The purchase and shopping cart buttons can only be viewed after login.
